Introduction à HTML5

HTML est un langage de balisage standard, qui signifie Hyper Text Markup Language. C’est un langage largement utilisé pour créer des pages Web. HTML inventé par Tim Berners-Lee à la fin de 1991, mais sa première version « HTML 1.0 » a été publiée en 1993, et « HTML 2.0 » a été la première spécification HTML standard, qui a été publiée en 1995.

Qu’est-ce que le HTML ?

HTML (HyperText Markup Language) est un langage de balisage standard pour concevoir la structure d’une page Web. HTML est la combinaison de l’hypertexte et du langage de balisage. L’hypertexte définit les liens internes entre les pages Web, et le langage de balisage définit la mise en page et la présentation du texte et des médias.

Structure HTML de base

La structure de base d’un document HTML contient les quelques balises obligatoires qui doivent être utilisées pour créer une page Web. Cette structure doit être suivie pour créer tout type de document HTML.
Éléments de la structure de base HTML

Voici les balises de base qui définissent la structure HTML de base :

1. <!DOCTYPE html>

Cet élément définit le type de document en tant que HTML. Cet élément doit être écrit avant la rédaction de tout document HTML.

2. <html>...</html>

La balise <html> est la balise parent de tous les éléments HTML.

3. <head>...</head>

La balise <head> est un conteneur pour les éléments qui ne sont pas affichés sur la page.

4. <title>...</title>

Définit le titre de la page visible dans l'onglet du navigateur.

5. <body>...</body>

Contient le contenu principal affiché dans le navigateur.

6. <h1>...</h1>

Définit un titre principal.

7. <p>...</p>

Définit un paragraphe de texte.

Tutoriel CSS

Le CSS (Cascading Style Sheets) est un langage utilisé pour styliser les pages HTML. Il permet de définir la mise en page, les couleurs, les polices, et bien plus encore.

Qu’est-ce que le CSS ?

CSS est l’acronyme de « Cascading Style Sheet ». Il s’agit d’un langage de feuille de style utilisé pour décrire la présentation d’un document écrit dans un langage de balisage comme le HTML. CSS aide les développeurs Web à contrôler la mise en page et d’autres aspects visuels des pages Web. CSS joue un rôle crucial dans le développement Web moderne en fournissant les outils nécessaires pour créer des sites Web visuellement attrayants, accessibles et réactifs.

Avantages de l’utilisation de CSS

Conception réactive - CSS offre des fonctionnalités telles que les requêtes multimédias qui permettent aux développeurs de créer des mises en page réactives qui s’adaptent à différentes tailles d’écran et appareils, garantissant ainsi une expérience utilisateur cohérente.

Flexibilité et contrôle - CSS offre un contrôle précis de la présentation des éléments HTML, permettant aux développeurs de personnaliser la mise en page, la typographie, les couleurs et d’autres propriétés visuelles.

Cohérence et réutilisabilité - Les développeurs peuvent assurer la cohérence de l’ensemble du site Web, en définissant des styles dans un fichier CSS central. Les styles peuvent être réutilisés sur plusieurs pages, ce qui réduit la redondance et facilite les mises à jour.

Optimisation des moteurs de recherche (SEO) - CSS peut être utilisé pour structurer le contenu de manière à améliorer la visibilité des moteurs de recherche.

Facilité de maintenance - Les fichiers CSS centralisés facilitent la maintenance et la mise à jour des styles d’un site Web. Les changements peuvent être appliqués à l’échelle mondiale, ce qui garantit l’uniformité et réduit le risque d’incohérences.

Chargement plus rapide des pages - Les fichiers CSS externes peuvent être mis en cache par les navigateurs, ce qui permet d’accélérer le chargement des pages pour les visites ultérieures sur un site Web. Ce mécanisme de mise en cache réduit la charge du serveur et la consommation de bande passante.

Composants de CSS

CSS fonctionne en associant des règles à des éléments HTML. Une règle CSS se compose de deux parties principales :

Un sélecteur qui spécifie le(s) élément(s) HTML à styliser.

Bloc de déclaration qui contient une ou plusieurs déclarations séparées par des points-virgules.

Chaque déclaration comprend un nom de propriété et une valeur, spécifiant l’aspect de la présentation de l’élément à contrôler.

📜 Flexbox & Grid Cheatsheet

  • Flexbox : display: flex;
  • Alignement horizontal : justify-content: space-between;
  • Alignement vertical : align-items: center;
  • Grid : display: grid;
  • Colonnes automatiques : grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  • Placement d'éléments : grid-column: 1 / span 2;

🎨 Astuces et hacks CSS

  • Alignement centré : display: flex; justify-content: center; align-items: center;
  • Texte en colonne : column-count: 2;
  • Effet de survol : :hover { transform: scale(1.1); }
  • Arrière-plan flou : backdrop-filter: blur(10px);
  • Texte en dégradé : background: linear-gradient(to right, red, blue); -webkit-background-clip: text; color: transparent;
